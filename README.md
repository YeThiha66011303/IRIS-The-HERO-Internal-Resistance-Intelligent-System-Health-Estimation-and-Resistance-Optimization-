Project Title

IRIS: The HERO
(Internal Resistance Intelligent System – Health Estimation and Resistance Optimization)

Project Overview

IRIS is a smart Battery Management System (BMS) designed for 4S–8S lithium battery packs. Besides providing conventional battery protection and monitoring functions, IRIS continuously measures the internal resistance of each battery cell and uses this information to estimate State of Health (SOH) and analyze cell-level degradation in real time.

Unlike conventional low-cost BMSs that rely mainly on voltage and current measurements, IRIS provides deeper insight into battery aging by monitoring changes in each cell's internal resistance. This allows early detection of degraded cells, improves maintenance decisions, and enhances battery safety and reliability.

Objectives
Design a reliable BMS for 4S–8S lithium battery packs.
Implement standard battery protection functions.
Measure the internal resistance, impedance of each battery cell in real time.
Estimate each cell's capacity and State of Health (SOH).
Analyze battery degradation at the individual cell level.
Provide real-time monitoring through a user interface.

Standard BMS Features
Individual cell voltage monitoring
Pack voltage monitoring
Charge and discharge MOSFET control
Over-voltage protection
Under-voltage protection
Cell voltage imbalance detection
Over-current protection
Short-circuit protection
Temperature monitoring
Passive or active cell balancing
Configurable protection parameters
Communication with MCU through SMBus/I²C

Innovative Features
1. Real-Time Internal Resistance Measurement
2. Cell-Level SOH Estimation
3. Cell Degradation Analysis
4. Intelligent Warning System
